    Private Functions

    Aquí se encuentran detalladas las funciones privadas utilizadas en la librería.

    ConstructArguments

    Aquí se encuentran detalladas las funciones para transformar un Cluster a un objeto de .NET.

    AtributosClusterToNet

    Esta función transforma un array de atributos (cluster) a un array de atributos (objeto .NET).

    Controls

    • ArrayDeDatos (array): atributos que se quieren transformar.
    • error in (error_cluster).

    Indicators

    • ArrayDeAtributos (array): atributos transformados a objetos de .NET.
    • error out (error_cluster).

    • error in (error_cluster).

    Indicators

    • arrayDeRespuestas (array): respuesta transformada y lista para ser usada.
