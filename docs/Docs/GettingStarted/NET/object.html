    EndpointConfig

    Enumerador utilizado a la hora de instanciar un nuevo objeto Wrapper para indicar que tipo de QTS_ServiceClient es.

    Este es un atributo propio del objeto Wrapper el cual sirve para establecer un tipo de conexión u otro.

    Warning

    Esta libreria esta pensada, desarrollada y testada como CoProcessor, en caso de usar HandHeld habrá que tener cuidado a la hora de realizar reportes.

    1
    public enum EndpointConfig { CoProcessor, HandHeld }

    Responses

    Estos objetos nos permiten tranformar las respuestas recibidas por el servicio, las cuales son bantante poco intuitivas e incluyen información innecesaria por unos objetos que incluyen aquellos datos necesarios e importantes para el usuario.

    Además los objetos respuesta del servicio serian inentilegibles por el usuario pues su definicion no es accesible por estos. Así pues, gracias a esta transformacion los usuarios pueden acceder a la informacion devuelta.
