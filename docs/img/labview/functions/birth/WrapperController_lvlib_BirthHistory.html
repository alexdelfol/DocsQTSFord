WrapperController.lvlib:BirthHistory.vi

Contiene las funciones relacionas con el BirthHistory

WrapperController.lvlib:UploadBirthHistory.vi

Función que reporta los datos del BirthHistory dada una pieza

Controles:

- basicInformation in: datos necesarios para hacer la subida.

- parametrosUploadBirthHistory: datos donde se incluye el identificador de la pieza a reportar entre otros

- datos: array de atributos que se quieren subir

- error in: Entrada de errores

Indicadores:

- basicInformtion out: salida de los datos basicos actualizados

- ArraydeRespuestas: respuesta obtenida del servidor

- error out: error producido al ejecutarse o el error de entrada si es que habia
