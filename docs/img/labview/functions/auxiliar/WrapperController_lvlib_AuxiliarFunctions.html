WrapperController.lvlib:checkIpAddress.vi

Función que comprueba si la cadena de caracteres de entrada es una direccion IP.

Controles:

- input string: Cadena de caracteres a comprobar

- error in: Entrada de errores

Indicadores:

- after match: Cadena devuelta, si no era una IP devuelve 0.0.0.0

- error out: Error producido durante el uso
